# WebDrivers Directory

Place your browser drivers here:

## Chrome Driver
- chromedriver.exe (version must match your Chrome browser)

## Edge Driver  
- msedgedriver.exe (version must match your Edge browser)

To find your Chrome version:
1. Open Chrome
2. Go to chrome://settings/help
3. Note the version number (e.g., 146.0.7680.165)

Download matching drivers from:
- Chrome: https://sites.google.com/chromium.org/driver/
- Edge: https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/
